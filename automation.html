Beschreibung

Konzeption und Implementierung einer skalierbaren, eventgetriebenen Automatisierungs-Infrastruktur auf Basis von n8n. Das System demonstriert die nahtlose Integration moderner Large Language Models (LLMs) in bestehende Systemlandschaften über das Model Context Protocol (MCP). Repetitive, fehleranfällige Prozesse der manuellen Datensichtung und -bewertung werden vollständig autonomisiert: von der API-gestützten Datenbeschaffung über eine KI-basierte Relevanz-Analyse bis hin zur strukturierten Bereitstellung im lokalen Workspace.
